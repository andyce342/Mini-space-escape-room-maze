# Mini-space-escape-room-maze
A mini text based maze game 

## 🚀 Intergalactic Escape Room
A blind-navigation survival game. You are the Commander, trapped in a high-security space station. With the lights out, you must rely on your suit's radar and your wits to find the way out.
------------------------------
## 🎮 How to Play

* Move: Type up, down, left, or right to feel your way through the dark.
* 📡 Radar: Your suit automatically scans for Walls (x) in every direction.
* 🚧 Security Gates: If you hit a locked gate (p), you must:
1. Hack the Door: Guess the correct access port (1, 2, or 3).
   2. Decipher the Terminal: Enter the password by identifying the Capitalized hint.
* 🏆 The Goal: Find the hidden Escape Pod (e) to launch to safety!

------------------------------
## 🛠️ Setup

   1. 💾 Make sure you have Python installed.
   2. 💻 Run the script in your terminal or VS Code.
   4. 👩‍🚀 Enter your name to begin the mission!

------------------------------
